# Catalogo Clarisse
Catalogo web de ropa para bebés organizado por categorías de edad (recien nacido, 3 meses, 6 meses, etc.)


## Tecnologías
- Xampp
- PHP
- MySQL
- Javascript
- CSS


## Instalación

1. Clonar el repositorio
git clone https://github.com/JosuePatinio/turing-catalogo-clarisse.git

2. Copiar el proyecto dentro de la carpeta htdocs de XAMPP
C:\xampp\htdocs\

3. Iniciar Apache y MySQL desde el panel de XAMPP.

4. Crear la base de datos
Abrir phpMyAdmin y crear una BD con nombre: catalogo_clarisse

5. Importar el archivo sql que contiene la BD, está en la carpeta backend del proyecto
*database/catalogo_clarisse.sql*

6. Acceder al proyecto desde el navegador
localhost/turing-catalogo-clarisse/frontend/login.php
